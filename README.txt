Twitter is an library project that to make request to fetch data from Twitter, and also defines User and Tweet class to structure the data.

Smashtag is the MVC project that user can type in the keywords and search tweets and twitters by using the functionalities in Twitter project. The searched data is stored into core data. And user can fetch images, URLs, and keywords statistics from core data.

To run the code on iPhone, 
	1. open TwitterStatistics.xcworkspace.
	2. select Smashtag project.
	3. select iPhone simulator or real device near upper-left corner of Xcode
	4. press run button near upper-left corner of Xcode.
